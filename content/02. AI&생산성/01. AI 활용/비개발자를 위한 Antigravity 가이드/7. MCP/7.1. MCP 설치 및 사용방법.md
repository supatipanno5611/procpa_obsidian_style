## 1. MCP란?
**MCP(Model Context Protocol)**[^1]는 AI 모델이 외부 데이터나 도구와 상호작용할 수 있도록 해주는 **개방형 표준 프로토콜**입니다.

언제부턴가 ChatGPT, Gemini, Claude 같은 AI 챗봇들이 내 이메일을 읽거나, 캘린더에 일정을 등록하는 등 다른 앱들과 자유롭게 연동되는 모습을 보셨을 겁니다. 과거에는 이런 기능을 구현하려면 각 서비스마다 별도의 '전용 플러그인'을 개발해야 했지만, MCP라는 **표준화된 규격** 하나로 이 모든 연결이 가능해졌습니다.

초기에는 MCP가 AI에게 전문적인 능력을 부여하는 역할까지 수행했으나, '스킬(Skills)'이 등장하게 되면서 현재 MCP는 **외부 데이터와의 '연결'** 이라는 본연의 기능 위주로 사용하게 되었습니다.

### 1.1. 외부 도구와의 연결
과거에는 AI에게 내 구글 캘린더 일정을 알려주려면 일일이 복사해서 붙여넣거나, 복잡한 전용 앱을 개발해야 했습니다. 하지만 MCP가 등장하면서 상황이 바뀌었습니다.

- **과거**: "오늘 내 일정 알려줘" -> (AI: 저에게는 권한이 없습니다.)
- **MCP 연결 후**: "오늘 내 일정 알려줘" -> (AI: 구글 캘린더를 확인해보니 오후 2시에 미팅이 있네요.)

### 1.2. 스킬(Skills)과의 차이점
지난 시간까지 알려드린 '스킬'과 헷갈리실 수 있습니다. 가장 큰 차이는 **'초점'** 에 있습니다.

- **스킬(Skills)**: AI가 **'내부적으로'** 특정 작업을 수행하는 능력을 키우는 데 집중합니다. (예: 블로그 글쓰기 규칙대로 글 작성하기, 특정 양식의 썸네일 만들기)
- **MCP**: AI를 **'외부 소프트웨어'** 와 연동하는 데 집중합니다. (예: Notion 페이지 읽어오기, Google Calendar 일정 확인하기)

## 2. MCP 서버 설치 방법
MCP를 사용하려면 Antigravity와 외부 앱을 연결해 주는 **'MCP 서버'** 를 설치해야 합니다. 크게 두 가지 방법이 있습니다.

### 2.1. MCP Store에서 설치
Antigravity 내장 스토어를 이용하는 가장 간단한 방법입니다.

1. Antigravity 우측 상단 `...` 메뉴 > `MCP Servers` 클릭
2. **"Discover"** 탭에서 원하는 서비스(예: **Notion**) 찾기
3. `Install` 버튼 클릭
4. 화면의 안내에 따라 계정 로그인 및 API 키 입력


> [!info]
> 현재(2026년 초 기준) MCP Store에 등록된 서비스는 아직 많지 않습니다. 하지만 Notion이나 GitHub 같은 주요 서비스는 아주 쉽게 연결할 수 있습니다.

### 2.2. Config 파일 수정해서 수동으로 설치
Store에 없는 서비스(예: NotebookLM 등)를 연결하려면 설정 파일을 직접 수정해야 합니다. 처음에는 낯설 수 있지만, 차근차근 따라 하면 어렵지 않습니다.

1. Antigravity 우측 상단 `...` 메뉴 > `MCP Servers` > `Manage MCP Servers` > `View raw config` 클릭
2. `mcpServers` 항목에 아래와 같은 형식으로 코드를 추가합니다.(예시 : 구글캘린더)

```json
"google-calendar": {
    "command": "npx",
    "args": [
    "-y",
    "@modelcontextprotocol/server-google-calendar"
    ],
    "env": {
    "GOOGLE_CLIENT_ID": "ID",
    "GOOGLE_CLIENT_SECRET": "SECRET"
    }
}
```

> [!tip]
> 이 과정이 어렵게 느껴진다면 늘 강조했듯 **AI에게 시키세요.**
> "나 구글캘린더 MCP 설치하고 싶은데 `config` 파일 어떻게 수정하면 돼?"라고 물어보고, 발급받은 API 키만 알려주면 AI가 알아서 코드를 작성해 줍니다.

## 3. MCP 서버 사용법
MCP 서버 설치가 완료되었다면, 사용 방법은 매우 간단합니다. **평소처럼 대화하면 됩니다.**

### 3.1. 자연어로 명령
별도의 복잡한 명령어 없이, AI에게 하고 싶은 말을 자연스럽게 하세요.

- "내 구글 캘린더에서 내일 일정에 추가해줘."
- "방금 작성한 내용 노션에 정리해줘."

AI는 여러분의 말을 이해하고, 필요한 MCP 도구를 스스로 판단하여 실행합니다.

### 3.2. 권한 허용
처음 MCP 도구를 사용할 때, Antigravity는 보안을 위해 **"이 도구를 실행해도 될까요?"** 라고 묻는 팝업을 띄웁니다.

이때 **[Allow Always]** 를 선택하면, 이후에는 묻지 않고 편리하게 사용할 수 있습니다.

## 4. 추천 MCP 서버
MCP 생태계는 매우 방대하지만, 비개발자(실무자)에게 가장 필요한 것은 **'내 업무 도구와의 연결'** 입니다. 복잡한 개발 도구보다는, 매일 사용하는 생산성 앱을 연결했을 때 그 진가를 발휘합니다.

따라서, **내 업무 환경에서 사용하는 앱의 MCP 서버**를 연결하는 것이 가장 중요합니다.

다음은 공통적으로 가장 추천하는 **3가지 MCP 서버**입니다.

### 4.1. Notion (노션) MCP
많은 분들이 업무 정리나 개인 메모용으로 Notion을 사용합니다. MCP를 연결하면 Antigravity가 Notion에 있는 방대한 자료를 직접 읽고, 수정하고, 정리해 줄 수 있습니다.

- **추천 대상**: Notion을 주력 노트 앱으로 사용하는 분
- **활용 예시**:
  - "내 '독서 리스트' 데이터베이스에서 아직 안 읽은 책 목록 뽑아줘."
  - "방금 이야기한 회의 내용, 내 '업무 일지' 페이지에 요약해서 추가해줘."
  - "노션에 있는 '프로젝트 가이드라인' 페이지 읽고, 그 규칙대로 기획안 초안 써줘."

### 4.2. Google Drive (구글 드라이브) MCP
MCP를 연결하면 내 구글드라이브에 있는 수많은 문서(PDF, Google Docs 등)를 AI가 직접 검색하고 내용을 읽어올 수 있습니다.

- **추천 대상**: 구글 드라이브를 메인 클라우드로 사용하는 분
- **활용 예시**:
  - "내 드라이브에서 '2025 사업계획서' PDF 찾아서 요약해줘."
  - "'프로젝트' 폴더에 있는 파일 리스트 보여줘."
  - "'회의록' 구글 docs 읽고, 주요 안건만 뽑아서 정리해줘."

### 4.3. NotebookLM - "나만의 지식 엔진" (강력 추천)
구글의 NotebookLM은 내가 가진 자료(PDF, 웹사이트 등)를 학습시켜 대화할 수 있는 도구입니다. 이를 MCP로 연결하면, **내가 가진 문서들을 근거로 정확하게 답변하는 AI**를 만들 수 있습니다. 가장 활용도가 높아 별도 챕터로 상세히 다루겠습니다.

> [!important]
> **MCP 서버 관리 팁**
> 스킬과 달리 MCP는 한꺼번에 너무 많은 도구를 설치하면 안정성 문제가 생기므로 **자주 쓰는 핵심 도구 3~4개**만 유지하는 것이 안정성 측면에서 좋습니다.
