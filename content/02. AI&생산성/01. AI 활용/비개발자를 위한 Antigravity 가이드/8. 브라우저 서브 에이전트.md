## 1. 브라우저 서브 에이전트란?

Antigravity 의 **브라우저 서브 에이전트**는 사용자를 대신해 웹 브라우저 (Chrome) 를 직접 제어하고 웹상의 다양한 작업을 수행할 수 있는 특화된 AI 입니다. 마치 내 컴퓨터에 앉아 마우스를 움직이고 키보드를 타이핑하는 개인 비서처럼 작동합니다.

### 1.1. 기능

- **다양한 브라우저 기능**: 특화된 시각 모델 (Gemini 2.5 Pro UI Checkpoint) 을 사용하여 **클릭, 스크롤, 텍스트 입력, 화면 캡처**, 등 사람과 동일한 방식으로 웹사이트에서 작업을 수행할 수 있습니다.
- **독립된 브라우징 환경**: 사용자의 일반적인 웹서핑과 에이전트의 활동을 격리하기 위해, 별도의 독립된 Chrome 프로필에서 확장 프로그램 형태로 실행됩니다. 에이전트가 브라우저 페이지를 제어하는 동안에는 창에 **파란색 테두리**가 표시되며, 현재 어떤 작업을 수행하고 있는지 설명하는 작은 패널이 나타납니다. (이 상태에서는 에이전트의 작업 혼선을 막기 위해 사용자의 직접 조작을 피해야 합니다.)
- **아티팩트를 통한 시각적 기록**: 작업 과정 전체를 비디오로 녹화하거나 상태 변화 전후의 스크린샷을 아티팩트 (Artifact) 로 남깁니다. 이를 통해 사용자는 브라우저 창을 계속 띄워놓고 감시할 필요 없이, 추후 생성된 동영상을 재생하여 작업 결과를 검증할 수 있습니다.

> [!note] 참고
> 브라우저 서브 에이전트는 본래 개발자들의 웹사이트 테스트를 위한 도구에 가까우나, 이를 응용하면 웹 브라우저에서 이루어지는 다양한 반복 업무를 자동화하는 데 유용하게 활용할 수 있습니다.

### 1.2. 사용방법

브라우저 서브 에이전트를 사용하는 방법은 매우 간단합니다.

1. **설정 활성화**: 먼저 Antigravity 의 Settings 화면에서 브라우저 서브 에이전트 기능을 활성화해 줍니다.

![Settings 화면](https://cdn.jsdelivr.net/gh/procpalee/my_obsidian_images/img/20260228124418719.png)

2. **브라우저 열기 및 확장 프로그램 설치**: 화면 우측 상단에 있는 `Open Browser` 버튼을 클릭하시면 새로운 크롬 브라우저 창이 열립니다. 이후 브라우저 화면에 나타나는 안내에 따라 Chrome Extension(확장 프로그램) 을 설치해 주시면 모든 준비가 끝납니다.

![Open Browser 화면](https://cdn.jsdelivr.net/gh/procpalee/my_obsidian_images/img/20260228124537457.png)

3. **작업 요청하기**: 설정이 끝나면 평소처럼 채팅창에서 자연어 텍스트로 브라우저 작업을 요청할 수 있습니다.

- " 브라우저를 열어줘 "
- " 구글 홈페이지에 접속해줘 "
- " 웹 브라우저를 켜서 [웹사이트 주소] 로 들어가줘 "

### 1.3. 한계점 및 권장 사용 케이스

브라우저 서브 에이전트는 사람처럼 브라우저를 직접 조작하는 데 특화되어 있습니다. 따라서 짧은 시간에 수천, 수만 개의 웹페이지 데이터를 긁어오거나 (크롤링), 복잡한 데이터를 대량으로 수집하는 작업에는 **적합하지 않습니다**. 이러한 대량 데이터 수집 작업에는 파이썬 라이브러리를 활용하는 것이 훨씬 빠릅니다.

반면, 다음과 같은 상황에서는 브라우저 서브 에이전트를 사용하는 것이 유리합니다.

- **비정형 문서 스크랩**: 대량은 아니지만 1~2 개 페이지 분량의 기사, 블로그 글, 특정 게시판 공지사항 등을 읽어오고 요약해야 할 때
- **동적 UI 상호작용 필수 작업**: 단순한 조회 성격을 넘어 ' 버튼 클릭 ', ' 폼 입력 ', ' 체크박스 선택 ' 등 화면 내 요소와 상호작용해야만 결과를 얻을 수 있는 웹사이트를 다룰 때
- **시각적 기록 수집 및 검수**: 진행 과정 전체나 특정 결과 페이지를 동영상 녹화본과 스크린샷 아티팩트로 남겨, 시각적인 보고서 형태나 검수가 필요할 때

## 2. 브라우저 서브 에이전트 활용 사례

개발 지식이 전혀 없더라도, 자연어 지시만으로 브라우저 상에서 이루어지는 귀찮고 반복적인 업무를 자동화할 수 있습니다. 다음은 실무에서 바로 적용해 볼 수 있는 구체적인 활용 케이스입니다.

### 2.1. 사례 1 : 기업 공시자료 조회

### 2.2. 사례 2 : 게시판 자료 모니터링 및 요약
