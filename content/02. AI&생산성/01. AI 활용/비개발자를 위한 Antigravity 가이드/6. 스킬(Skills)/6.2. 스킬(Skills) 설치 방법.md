## 1. 스킬 위치

스킬을 설치하기 전에, 이 파일들이 어디에 저장되는지를 먼저 알아야 합니다. 
Rules, Workflow와 마찬가지로 Antigravity는 스킬을 **전역(Global)** 과 **워크스페이스(Workspace)** 두 가지 범위로 관리합니다.

### 1.1. 전역 스킬 (Global Skills) : 
내 컴퓨터에서 수행하는 **모든 프로젝트**에서 공통적으로 사용하고 싶은 도구입니다. 

- **저장 경로** : `C:\Users\[사용자명]\.gemini\antigravity\skills\`
- **적합한 스킬 예시**
  - **skill-creator**: 새 스킬을 만들거나 기존 스킬을 업데이트할 때 필수적이므로 전역으로 두는 것이 유리합니다.
  - **document-skills (DOCX, PDF, PPTX, XLSX)**: 문서 생성 및 변환 도구로, 어떤 프로젝트에서든 문서 작업은 필요하므로 전역으로 두는 것이 유리합니다.

### 1.2. 워크스페이스 스킬 (Workspace Skills)
지금 작업 중인 **특정 프로젝트**에서만 유효한 스킬입니다.

- **저장 경로** : `[현재 프로젝트 폴더]\.agent\skills\`
- **적합한 스킬 예시**
  - **brand-guidelines**: 특정 프로젝트만의 로고, 컬러, 폰트 규정 적용에 사용합니다.
  - **startup-financial-modeling**: 전반적인 엑셀 작업이 아닌 모델링 작업에 특화된 스킬이므로 워크스페이스에 두는 것이 유리합니다.

> [!note] 스킬 탐색 우선순위
> 전역 스킬과 워크스페이스 스킬은 탐색 우선순위가 있습니다. 에이전트는 먼저 **워크스페이스 (.agent/skills)** 폴더를 확인하고, 해당 폴더에 스킬이 없을 경우에만 **전역 (.gemini/antigravity/skills)** 폴더에서 스킬을 찾아 사용합니다.

## 2. 스킬 설치 방법

스킬을 설치하는 방법은 크게 세 가지가 있습니다.

### 2.1. GitHub 공식 저장소 활용 (수동 설치)
Anthropic(Claude 개발사)에서 제공하는 공식 스킬 저장소입니다. 가장 기본적이고 검증된 스킬들이 모여 있습니다.
![image.png](https://cdn.jsdelivr.net/gh/procpalee/my_obsidian_images/img/20260211135830620.png)
**1. 접속**: [Anthropic Skills GitHub](https://github.com/anthropics/skills) 에 접속합니다.

**2. 다운로드**: 우측 상단의 초록색 `<> Code` 버튼을 누르고 `Download ZIP`을 클릭하여 전체 파일을 받습니다.

**3. 복사**: 압축을 푼 뒤, `example-skills` 또는 `document-skills` 폴더 안에 있는 개별 스킬 폴더(예: `pdf`, `xlsx`)를 복사합니다.

**4. 설치**: 위에서 설명한 `skills` 폴더(`전역` 또는 `워크스페이스`)에 붙여넣습니다.

### 2.2. SkillsMP (마켓플레이스) 활용
[SkillsMP](https://skillsmp.com)는 Antigravity, Claude, Codex 등 다양한 에이전트에서 사용할 수 있는 25,000개 이상의 스킬을 모아둔 거대한 도서관입니다.
![image.png](https://cdn.jsdelivr.net/gh/procpalee/my_obsidian_images/img/20260211140357321.png)

**1. 접속**: [SkillsMP](https://skillsmp.com) 에 접속합니다.

**2. 검색**: 필요한 기능(예: `excel modeling`, `data analysis`)을 검색합니다.

**3. 다운로드**: 원하는 스킬 페이지의 우측 `$ downlaod --local` 에서 `wget skill.zip` 을 클릭하여 zip 파일을 내려 받습니다.
 
**4. 설치** : 압축을 푼 뒤 위에서 설명한 `skills` 폴더(`전역` 또는 `워크스페이스`)에 붙여넣습니다.

### 2.3. skills.sh 활용 (자동 설치)
`skills.sh`는 스킬을 검색하고 명령어 한 줄로 즉시 설치할 수 있는 CLI(터미널)[^1] 기반 마켓플레이스입니다. 파일을 직접 다운로드하고 옮기는 번거로움이 없어 가장 추천하는 방법입니다.
![image.png](https://cdn.jsdelivr.net/gh/procpalee/my_obsidian_images/img/20260211141052944.png)

**1. 탐색**: [skills.sh](https://skills.sh) 사이트에 접속합니다.

**2. 검색**: 필요한 스킬을 검색한 후 `명령어`를 복사합니다.

**3. 설치**: 터미널(명령 프롬프트)을 열고 아래 명령어를 입력합니다. 
- 폴더 경로는 탐색기 주소창에서 복사한 뒤 터미널에서 마우스 오른쪽 클릭으로 붙여넣습니다.
- 위의 검색 단계에서 복사한 명령어를 마우스 오른쪽 클릭으로 붙여넣습니다.

```bash
# 워크스페이스(특정 프로젝트)에 설치
cd "워크스페이스 폴더 경로"
npx skills add <owner/repo>

# 전역(모든 프로젝트 공통)으로 설치
npx skills add <owner/repo> -g
```

**4. 완료**: 명령어 입력 후 터미널에 나타나는 선택지에서 `Antigravity`를 선택(Enter)하고, 이후 안내에 따라 Enter를 눌러 설치를 마무리합니다.

**5. 예시**: `browser-use` 스킬을 설치하는 실제 과정입니다.

[^1]: **CLI (Command Line Interface)**: 마우스로 아이콘을 클릭하는(GUI) 대신, 키보드로 '명령어'를 입력하여 컴퓨터와 대화하는 방식입니다. 마치 영화 속에서 해커들이 검은 화면에 글자를 치는 것과 비슷하지만, 익숙해지면 훨씬 빠르고 정확하게 작업할 수 있습니다.
